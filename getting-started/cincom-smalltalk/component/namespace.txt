Next, we should create a name space for our work. Pick ''Class | New | Namespace...'' in the System Browser. In the dialog that opens, the package and Namespace fields are already correctly filled. Enter a name, which we will call WebCounter again. In the imports field, add ==Seaside.\*== so the field contents is: ==private Smalltalk.\* Seaside.\*==+namespace|width=40%+Then click OK. The ==WebCounter== name space is created and selected in the System Browser, and the definition is shown as:==Smalltalk defineNameSpace: #WebCounter ==    private: false ==    imports: 'private Smalltalk.* Seaside.*' ==    category: ''
