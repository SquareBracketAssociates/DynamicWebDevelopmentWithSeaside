Several properties of the Seaside environment can be set using the Settings Tool, accessible by picking ''Seaside | Settings'' in the Transcript. For each setting item there is online help provided. Click Help on the page with the setting item to view the description.+vw-settings|width=50%+  For example, you can change the port the server is running on by entering the new number in the Port: field and clicking ''Apply''. This stops the server and restarts it on the specified port.
