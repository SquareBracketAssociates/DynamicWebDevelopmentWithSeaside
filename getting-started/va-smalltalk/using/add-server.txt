Select ''Manage | Add adaptor ...'' to add an adaptor that will serve Seaside pages, see *ref:add-adaptor*. The added Server Adaptor will be in the stopped state.  +add-adaptor|width=60%+In the Add Server Adaptor dialog you can select the Server Adaptor class (currently only one class is available) and the port used to connect to the server, see *ref:add-adaptor-dialog*.+add-adaptor-dialog|width=60%+You can add multiple server adaptors which can listen on different ports.
